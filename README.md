# Sage (sage)
Sage provides cloud-based ERP, accounting, payroll, and HR software for businesses. The Sage Developer program provides APIs for integrating with Sage products including Sage Accounting, Sage Business Cloud, Sage Intacct, and Sage 200 to automate business processes and build integrations.
